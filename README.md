# 株式ポートフォリオ管理アプリ

yfinance を使った株式投資支援アプリ。日本株(銘柄コードに `.T` を自動付与)と米国株の両方に対応。
Streamlit のサイドバーで6つの機能を切り替えて使用します。

## 機能

1. **ポートフォリオ管理** — 保有銘柄の登録/編集/削除、評価額・損益の自動計算、資産配分の円グラフ
2. **株価チャート** — ローソク足、移動平均線(25/75日)、ボリンジャーバンド、出来高
3. **スクリーニング** — 銘柄リストに対し PER/PBR/配当利回り/時価総額を取得し条件で絞り込み
4. **配当金シミュレーター** — 配当履歴から年間配当金を試算、増配/減配傾向をグラフ化
5. **アラート・ウォッチリスト** — 目標株価の設定と条件到達銘柄のハイライト
6. **バックテスト** — 移動平均クロス戦略のシミュレーション、Buy&Hold との比較
7. **株価予測** — GBM/ブートストラップ/線形トレンド/平均回帰の4モデルでファンチャート表示
8. **ニュース・決算** — 関連ニュース見出しと次回決算予定日
9. **銘柄比較** — 2〜4銘柄の正規化リターン重ね描きと指標の横並び比較
10. **リスク指標** — シャープレシオ・最大ドローダウン・年率ボラティリティ・ベータ

※ 国内銘柄は同梱の `上場企業一覧.csv` から企業名・コード・市場区分で検索可能(大文字小文字・全角半角を区別しない)。

## セットアップ

```bash
# 仮想環境(任意)
python -m venv .venv
source .venv/bin/activate

# 依存ライブラリのインストール
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

ブラウザが自動で開きます(既定 http://localhost:8501)。

## 銘柄コードの入力方法

- **日本株**: 数字のみ(例: `7203`)→ 自動で `7203.T` に変換
- **米国株**: ティッカー(例: `AAPL`, `MSFT`)

## データについて

- 株価データは [yfinance](https://github.com/ranaroussi/yfinance)(非公式ライブラリ)由来で、**15〜20分程度の遅延**があります。
- 取得失敗時は各機能でエラーメッセージを表示します。投資判断は自己責任で行ってください。
- 取得結果は `@st.cache_data` でキャッシュ(5〜60分)し、過度なリクエストを抑制します。

## データ保存

保有銘柄・ウォッチリストは `data/` 配下の JSON に保存されます(Git管理対象外)。

## ディレクトリ構成

```
app.py              # エントリポイント(サイドバーでページ切替)
features/           # 各機能
  portfolio.py  chart.py  screening.py  dividend.py  watchlist.py  backtest.py
utils/
  data_fetch.py     # yfinanceラッパー(キャッシュ+エラーハンドリング)
  indicators.py     # テクニカル指標
  storage.py        # JSON永続化
data/               # 永続化データ(JSON)
```
